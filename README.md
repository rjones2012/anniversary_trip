# The Grand Tour — Summer 2026

Single-file trip site (London · Costa Navarino · Champagne · Paris).

## Publish
1. Create a new GitHub repo.
2. Upload `index.html` (and `vercel.json`) → Commit.
3. vercel.com → Add New → Project → import the repo → Deploy.
4. Get your `*.vercel.app` URL. Open in Safari → Share → Add to Home Screen.

## Edit later
Change `index.html`, commit — Vercel redeploys automatically.

Checklist state saves per-device in the browser (no account needed).
